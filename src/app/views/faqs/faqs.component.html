What is the purpose of this app?

This app stores pictures and data related to those pictures. It can be very useful for tracking images. For example, a use case for this is to keep track of images of failed automated tests. Using this app you exactly see

  • The name of the test
  • The page where you where
  • The action you were performing
  • The time when the failure happened
  • The picture of the page when it failed

This will help the QA Engineers debug test failures quicker and report bugs to the dev team faster.
